Driver Drowsiness Detection using YOLO

Features

  • Real-time eye aspect ratio (EAR) calculation
  • Facial landmark detection using OpenCV and optimized deep learning networks
  • High performance on edge devices via optimized YOLO weights
  • Configurable alarm threshold and audio trigger mechanisms

Challenges Solved

Managing false positives in varying lighting conditions (e.g., night-time driving). Integrated a rolling average filter over sequential frames to ensure alerts are only triggered during prolonged eye closures.

Results & Metrics

Maintained a steady 30 FPS inference rate on standard mobile and edge CPUs, achieving 98% detection accuracy evaluated on a heavily varied driver dataset.
